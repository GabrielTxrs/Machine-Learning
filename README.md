# Para executar esse projeto siga os passos abaixo:

### 1. Abrir terminal na pasta ATIVIDADE-1
```
$ cd ATIVIDADE-1
```

### 2. Ativar Ambiente Virtual
```
$ .\venv\Scripts\activate
```

### 3. Instalar Pacotes
```
$ pip install -r requirements.txt
```

### 4. Iniciar o Jupyter Notebook
```
$ jupyter notebook
```
